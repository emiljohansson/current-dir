# current-dir [![Build Status](https://travis-ci.org/emiljohansson/current-dir.svg?branch=master)](https://travis-ci.org/emiljohansson/current-dir) [![npm version](https://img.shields.io/npm/v/current-dir.svg)](https://www.npmjs.com/package/current-dir)

> Returns the full path to the current directory.

## Install

```
$ npm install --save current-dir
```

## Usage

```js
var currentDir = require('current-dir');
currentDir();
// => /Users/username/
```

## API

### currentDir()

Returns the path to the current directory.
